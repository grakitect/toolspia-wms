# 간이 재고관리 프로그램 (웹)

회사 내부 소규모(약 20명) 사용을 가정한 기본형 재고관리 시스템입니다.

## 포함 기능

- 대시보드 (기본 지표)
- 기본상품정보 (단건 등록/수정, 엑셀 업로드)
- 실재고현황
- 입고 (단건/엑셀 업로드, 미등록 상품 제한, 입고처 등록/선택)
- 출고 (단건/엑셀 업로드, 미등록 상품 및 재고부족 제한, 출고처 등록/선택)
- 재고조정 (+/- 조정)
- 입출고이력 (최신순 조회, 상품 검색)
- 알림 (향후 확장용 플레이스홀더)
- 거래처 마스터 관리 (입고처/출고처/구매처)

## 실행 방법

1. Node.js 18+ 설치
2. 프로젝트 폴더에서 아래 명령 실행

```bash
npm start
```

3. 브라우저에서 `http://localhost:3000` 접속

## 데이터 저장

- `data/db.json` 파일에 저장됩니다.
- 백업 시 해당 파일을 주기적으로 복사하면 됩니다.

## 엑셀 업로드 템플릿

### 상품 업로드

권장 컬럼 순서(구글시트 기준):

- `품목코드(이카운트)`
- `바코드`
- `물류바코드`
- `품목명(이카운트)`
- `상태`
- `판매처` (쉼표로 다중 입력 가능)
- `판매처관리코드`
- `판매처 품목명`
- `규격`
- `구매처`
- `수급형태`
- `발주부서`
- `발주담당자` (쉼표로 다중 입력 가능)
- `구매처 품목코드`
- `구매처 품목명`
- `창고그룹`
- `구분`
- `카테고리` (쉼표로 다중 입력 가능)

### 입고/출고/조정 업로드

필수 컬럼:

- `productCode`
- `qty`

선택 컬럼:

- `partner` (입고처/출고처)
- `memo`
- `user`

> 첫 시트 기준으로 읽습니다. xlsx/xls/csv 지원.
> 각 업로드 카드의 "가이드 다운로드" 버튼으로 샘플 파일 생성 가능.
